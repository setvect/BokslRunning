# AGENTS.md

## 목적
- 이 문서는 초반 바이브 코딩을 위한 최소 작업 규칙만 정의한다.
- 상세 요구사항/화면/정책은 아래 기준 문서를 우선 참조한다.

## 기준 문서
- `docs/PRD.md`
- `docs/storyboard.md`
- `docs/tech-stack.md`
- `docs/implementation-plan.md`

## 작업 원칙
- MVP 범위 안에서만 구현한다.
- 구현 순서는 `docs/implementation-plan.md`의 Phase를 따른다.
- 큰 변경 시 해당 Phase 체크박스를 갱신한다.
- 문서와 코드가 충돌하면 PRD 기준으로 맞춘다.
